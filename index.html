¿Que es Javascript?

JavaScript es un popular lenguaje de programación de scripts para añadir funcionalidades interactivas y otros contenidos web dinámicos a las páginas web. Algunos ejemplos conocidos de contenido en JavaScript son los formularios rellenables, las presentaciones de galerías de fotos y los gráficos animados.Es un excelente punto de partida para aquellos que buscan aprender más sobre el desarrollo de sitios web.
